# Vic3 mapdata editor

## 简介

Vic3 mapdata editor是基于[Jomini]("https://github.com/nickbabcock/jomini")的, 开源的Vic3地图省份编辑器。

## 特性

+ ✅可视化编辑Vic3的省份(含region_state)
+ ✅可视化编辑Vic3的省份资源、城市地块、人口、建筑
+ ✅可视化编辑Vic3的战略地区
